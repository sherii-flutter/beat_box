# 🎵 Music Player App

A beautiful and modern **Music Player Application** built with **Flutter & Dart**.  
This app allows users to browse songs, search music, play audio, and manage their favorite tracks with a smooth and attractive UI.

## ✨ Features

- 🎵 Play local music files
- ▶️ Play and pause songs
- 🔍 Search songs by name or artist
- ❤️ Add songs to favorites
- 🔄 Shuffle and repeat options
- 🎧 Modern dark theme UI
- 📱 Responsive Flutter design
- 🎶 Smooth audio playback experience

---

## 🛠️ Technologies Used

- **Flutter**
- **Dart**
- **Material UI**
- **Just Audio Package**
- **Shared Preferences**
- **Audio Player Integration**
