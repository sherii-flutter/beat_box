# Beat Box 🎵

A Flutter Beat Box application that allows users to play different beat sounds with a clean and responsive user interface.

## ✨ Features

- Play multiple beat sounds
- Simple and attractive UI
- Fast performance
- Responsive design

## 🛠️ Built With

- Dart
